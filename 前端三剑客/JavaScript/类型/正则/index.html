正则表达式是用于匹配字符串中字符组合的模式。在 JavaScript中，正则表达式也是对象。这些模式被用于 RegExp 的 exec 和 test 方法, 以及 String 的 match、matchAll、replace、search 和 split 方法。

()是匹配，[]是条件筛选的字符集合模式。

需求 1：获取 cookie 中的 o2o_shop_user_name 参数。

需求 2：目标字符串由四个部分组成：职位层级、职位描述、起始层级、结束层级，如'M8(战略管理63-66)'。

需求3：将字符串转成千分位。例如 ‘12345678’ 转化成千分位是 ‘12,345,678’。(零宽断言)
